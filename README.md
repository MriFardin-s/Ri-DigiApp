project Ri-DigiApp

Description:

Ri-DigiApp is modern, high-performance digital service platform designed to provide a easy browsing and shopping experience for digital tools and subscriptions plans. Built with a "user-first" approach, the application allows users to explore various product tiers, compare features, and manage their selections through an interactive real-time cart system.

Technology that I use:

The project made by modern tech stack to ensure speed, maintainability,

React.js (Vite): Utilized for building a fast, component-based user interface with efficient rendering.

Tailwind CSS: Employed for a utility-first styling approach, ensuring a highly customized and responsive UI.

Lucide React & React Icons: Integrated to provide a consistent and professional visualization through high-quality icon.

3 Key features:

1. Dynamic Tab-Based Navigation & UI Switching
The application features an intelligent layout ingine that toggles between landing page and a dedicated cart view. When the "Products" tab is active, the app renders full marketing sections (Hero, Stats, Pricing). Switching to the "Cart" tab dynamically hides these sections to provide a focused, distraction-free checkout environment.

2. State-Synced Real-Time Cart
By implementing "State Lifting," the application maintains a single "source of truth" for the shopping cart. This ensures that when a user clicks "Buy Now" on a product card, the navigation bar's cart count and the hidden Cart Section are updated instantly without requiring a page reload.

3. Advanced Responsive Grid Layouts
The UI is engineered with CSS Grid and Flexbox to ensure a flawless experience across all devices. The project features a complex responsive footer and product grid that automatically rearranges from a single-column layout on mobile to a sophisticated multi-column grid on large desktop monitors.
